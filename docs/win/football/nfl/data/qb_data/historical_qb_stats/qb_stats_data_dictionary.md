# Quarterback Master Dataset - Data Dictionary

This document serves as a reference for all columns in the merged `qb_stats_{season}.csv` files. The dataset is a combination of standard season-long statistics (from FantasyPros) and advanced passing metrics. 

> [!NOTE]
> Any column ending in `_adv` represents data pulled from the advanced stats source that shared a name with a standard FantasyPros column. Where duplicates existed, the `fp_qb` column is the primary metric, and the `_adv` column is retained for reference/comparison.

## Identifiers & Basics
| Column Name | Source | Description |
| :--- | :--- | :--- |
| `season` | Both | The NFL season year (e.g., 2021). |
| `team` | FP | The player's team abbreviation (Standardized). |
| `team_adv` | Adv | The player's team abbreviation according to the advanced stats source. |
| `qb_name` | Both | The Quarterback's full name (Standardized). |
| `fantasypros_player_raw` | FP | The raw string identifier directly from FantasyPros (e.g., "Josh Allen BUF"). |
| `rank` | FP | The player's FantasyPros end-of-season rank among Quarterbacks. |
| `position` | Adv | The player's roster position (Filtered to QB/WR/etc. containing 'QB'). |
| `age` | Adv | The player's age during that season. |
| `awards` | Adv | Any notable awards won by the player that season (e.g., PB, AP). |

## Game Participation
| Column Name | Source | Description |
| :--- | :--- | :--- |
| `games` | FP | Total games played in the season. |
| `games_adv` | Adv | Total games played according to the advanced source. |
| `starts` | Adv | Total games started at Quarterback. |
| `plays` | Adv | Total offensive snaps/plays the Quarterback was involved in. |

## Standard Passing Stats (FantasyPros Base)
| Column Name | Source | Description |
| :--- | :--- | :--- |
| `completions` | FP | Total passes completed. |
| `attempts` | FP | Total pass attempts. |
| `completion_pct` | FP | Base completion percentage. |
| `passing_yards` | FP | Total passing yards for the season. |
| `yards_per_attempt` | FP | Passing yards divided by total attempts. |
| `passer_rating` | FP | Standard NFL Passer Rating. |
| `red_zone_attempts` | FP | Pass attempts made from inside the opponent's 20-yard line. |
| `adjusted_completion_pct` | FP | Completion percentage adjusted for drops, throwaways, and spikes. |

## Deep Passing & Explosive Plays
| Column Name | Source | Description |
| :--- | :--- | :--- |
| `air_yards` | FP | Total distance the ball traveled in the air past the line of scrimmage on all passes. |
| `air_yards_per_attempt` | FP | Average air yards per pass attempt. |
| `passes_10_plus_yards` | FP | Number of completed passes gaining 10+ yards. |
| `passes_20_plus_yards` | FP | Number of completed passes gaining 20+ yards. |
| `passes_30_plus_yards` | FP | Number of completed passes gaining 30+ yards. |
| `passes_40_plus_yards` | FP | Number of completed passes gaining 40+ yards. |
| `passes_50_plus_yards` | FP | Number of completed passes gaining 50+ yards. |

## Advanced Passing Metrics (Air Yards & YAC)
| Column Name | Source | Description |
| :--- | :--- | :--- |
| `intended_air_yards` | Adv | Total depth of target across all pass attempts, regardless of completion. |
| `intended_air_yards_per_att` | Adv | Average depth of target per pass attempt. |
| `completed_air_yards` | Adv | Total air yards strictly on *completed* passes. |
| `completed_air_yards_per_comp` | Adv | Average air yards per completed pass. |
| `completed_air_yards_per_att` | Adv | Average completed air yards divided by total attempts. |
| `yac` | Adv | Total Yards After Catch generated by the receivers on the QB's passes. |
| `yac_per_comp` | Adv | Average Yards After Catch per completion. |

## Passing Accuracy & Mistakes
| Column Name | Source | Description |
| :--- | :--- | :--- |
| `poor_throws` | FP | Pass attempts deemed uncatchable or poorly thrown. |
| `bad_throws` | Adv | Pass attempts deemed uncatchable. (Counterpart to `poor_throws`). |
| `bad_throw_pct` | Adv | Percentage of attempts deemed uncatchable. |
| `on_target` | Adv | Pass attempts that hit the receiver in stride or in the numbers. |
| `on_target_pct` | Adv | Percentage of attempts deemed on target. |
| `drops` | FP | Number of passes dropped by receivers. |
| `drops_adv` | Adv | Drops as tracked by the advanced stats provider. |
| `drop_pct` | Adv | Percentage of catchable passes that were dropped. |
| `batted_passes` | Adv | Passes batted down at the line of scrimmage. |
| `throwaways` | Adv | Passes intentionally thrown away to avoid a sack. |
| `spikes` | Adv | Passes spiked to stop the clock. |

## Pocket Presence, Pressure & Sacks
| Column Name | Source | Description |
| :--- | :--- | :--- |
| `pocket_time` | FP | Average time (in seconds) the QB spends in the pocket between the snap and throwing/scrambling. |
| `pocket_time_adv` | Adv | Pocket time as tracked by the advanced stats provider. |
| `blitzes` | FP | Number of dropbacks where the defense blitzed. |
| `times_blitzed` | Adv | Advanced tracker for defensive blitzes. |
| `hurries` | FP | Number of times the QB was forced to throw earlier than intended due to pressure. |
| `hurries_adv` | Adv | Advanced tracker for hurries. |
| `knockdowns` | FP | Number of times the QB was knocked down after throwing the ball. |
| `hits_allowed` | Adv | Total QB hits allowed. |
| `pressures` | Adv | Total pressures (hurries + knockdowns + sacks). |
| `pressure_pct` | Adv | Percentage of dropbacks where the QB was pressured. |
| `sacks` | FP | Total times sacked. |
| `pressure_to_sack_rate` | FP | The percentage of pressures that ultimately resulted in a sack. |

## Rushing & Scrambling
| Column Name | Source | Description |
| :--- | :--- | :--- |
| `rush_attempts` | Adv | Total rushing attempts (designed runs + scrambles). |
| `rush_yards` | Adv | Total rushing yards. |
| `scrambles` | Adv | Number of times the QB scrambled out of the pocket to run. |
| `yards_per_scramble` | Adv | Average yards gained on scramble plays. |
| `yards` | Adv | Total combined offensive yards (Passing + Rushing) according to the advanced provider. |
